Expand description

An enum of all the different kinds of wasm instructions.

Note that the #[walrus_expr] macro rewrites this enum’s variants from

ⓘ
enum Instr {
    Variant { field: Ty, .. },
    ...
}

into

ⓘ
enum Instr {
    Variant(Variant),
    ...
}

struct Variant {
    field: Ty,
    ...
}
